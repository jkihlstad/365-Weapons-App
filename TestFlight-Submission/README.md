## Quick Submission Checklist

### Before Archiving
- [ ] Update version number in Xcode (General > Identity)
- [ ] Update build number in Xcode
- [ ] Ensure all API keys are removed from code (using secure storage)
- [ ] Test on real device
- [ ] Run all tests

### In App Store Connect
- [ ] Create new app or version
- [ ] Fill in app description
- [ ] Upload screenshots (all required sizes)
- [ ] Set pricing (Free)
- [ ] Set content rating
- [ ] Add privacy policy URL
- [ ] Submit for review

### Archive & Upload
1. In Xcode: Product > Archive
2. Window > Organizer
3. Distribute App > App Store Connect
4. Upload
