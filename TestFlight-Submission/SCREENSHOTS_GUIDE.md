# Screenshot Requirements for App Store

## Required Screenshot Sizes

### iPhone
- **6.7" Display (iPhone 15 Pro Max):** 1290 x 2796 pixels
- **6.5" Display (iPhone 14 Plus):** 1284 x 2778 pixels
- **5.5" Display (iPhone 8 Plus):** 1242 x 2208 pixels

### iPad (if supporting)
- **12.9" Display (iPad Pro):** 2048 x 2732 pixels

## Recommended Screenshots (in order)

1. **Dashboard** - Show the main dashboard with metrics
2. **Orders List** - Display order management view
3. **Product Creation** - Show the create product form
4. **AI Chat** - Demonstrate the AI assistant
5. **Codes Management** - Show discount codes feature
6. **Settings/Theme** - Show light/dark mode options

## Screenshot Tips

- Use the iOS Simulator to capture perfect screenshots
- Capture in both Light and Dark modes
- Don't include status bar time (use Xcode simulator features)
- Use clean, professional test data
- Highlight key features with callouts if needed

## How to Capture Screenshots

### Using Simulator:
1. Run app in Simulator
2. Navigate to the screen
3. Press Cmd+S to save screenshot
4. Screenshots save to Desktop

### Using Xcode:
1. Debug > Simulate Screenshot

## App Preview Video (Optional)

- Up to 30 seconds
- 1080p or 4K resolution
- Show key features in action
- No audio required but recommended
