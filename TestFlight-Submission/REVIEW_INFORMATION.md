This is an admin dashboard app for the 365 Weapons e-commerce platform. It allows business administrators to:

1. View dashboard analytics and metrics
2. Manage customer orders
3. Create and edit products
4. Manage vendor partners and commissions
5. View customer inquiries and messages
6. Create and track discount codes
7. Use AI-powered chat for business insights

The app requires an admin account to access. Please use the demo credentials provided above.

The app connects to our backend API at:
- Convex backend for data
- OpenRouter for AI chat features
- OpenAI for voice features

All features are fully functional with the demo account.
