## Description (4000 characters max)

365 Weapons Admin is the official companion app for managing your 365 Weapons e-commerce business. Built for administrators and business owners, this powerful dashboard puts complete control of your firearms business in your pocket.

**Key Features:**

DASHBOARD & ANALYTICS
- Real-time business metrics and KPIs
- Revenue tracking and trends
- Order status overview
- Quick action shortcuts

ORDER MANAGEMENT
- View and manage all customer orders
- Update order status and tracking
- Process refunds and returns
- Bulk order operations

PRODUCT CATALOG
- Create and edit products with variants
- Manage pricing and inventory
- Upload product images and videos
- Set product categories and page placement

VENDOR MANAGEMENT
- Onboard new partner stores
- Track vendor commissions
- Manage discount codes
- Monitor partner performance

CUSTOMER COMMUNICATION
- View service inquiries
- Respond to contact submissions
- Manage newsletter subscribers
- Track vendor applications

DISCOUNT CODES
- Create promotional codes
- Set percentage or fixed discounts
- Configure partner commissions
- Track code usage

AI-POWERED ASSISTANT
- Get intelligent business insights
- Natural language queries
- Voice command support
- Smart recommendations

APPEARANCE
- Light, Dark, and Auto modes
- Auto-switches based on time of day
- Customizable schedule

SECURITY
- Secure authentication
- API keys stored in iOS Keychain
- Role-based access control

Built with SwiftUI for a native iOS experience. Requires a 365 Weapons backend account.
